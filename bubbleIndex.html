Bubble Magnet Controller

So, I was looking for inspiration for something to do that is interactive on my portfolio and came across this. I really enjoyed it, and saw it used Canvas which I feel a little weak on, so I re-created it to get some canvas experience. I also added a few options to make it a little bit better as it also allowed me to better understand everything that was happening.

I ultimately decided not to use it in my portfolio, but I had already converted it to a JS file that can easily be imported into a project for use, so I thought why not post it. In accordance with CodePen license, this follows the same MIT License policy of being open sourced.

View the JS here: Link

Simply put this in your head or at the end of your html file : <script src="bubbleCanvas.js"></script> and move the JS file to your project. This also requires jQuery so make sure the file is included.

Start the script by calling startBubbling('body', true);

Replace body with the class or id call, i.e. '.center-box' or '#middle', quotation is required

If you dont want the controls to appear, replace true with false. Edit the css of the controls using class magnetControls. Edit the JS file under the function startBubbling to alter any of the controls.

Click on any of the below bordered areas to start bubbling inside.

Click again to attract or push the bubbles away.

Automatically fits any size and shape div

Simply reshape using CSS. The application applies a rectangular canvas, so in shaped divs, some of the canvas will be cut off.

Or you can have it fill the screen by keeping body and adding the following CSS rule :

canvas { background: [add your color option]; display: block; left: 0; position: absolute; top: 0; }

Or make a div height: 100vh; width: 100%;


And it reshapes automatically too
